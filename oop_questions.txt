1. Describe the difference between a Class and an Object.
Object - Objects have states and behaviors. Example: A dog has states - color, name, breed as well as behaviors -wagging, barking, eating. An object is an instance of a class.

 Class - A class can be defined as a template/blue print that describes the behaviors/states that object of its type support. A class is a blue print from which individual objects are created
